## ভূমিকা

### BuildBarguna কী?

BuildBarguna একটি গ্রুপ ইনভেস্টমেন্ট প্ল্যাটফর্ম যেখানে ব্যবহারকারীরা বিভিন্ন প্রজেক্টে শেয়ার কিনে বিনিয়োগ করতে পারে এবং মাসিক লভ্যাংশ উপার্জন করতে পারে। এছাড়াও প্ল্যাটফর্মে টাস্ক সম্পন্ন করে পয়েন্ট অর্জন এবং রিওয়ার্ড রিডিম করার সুযোগ রয়েছে।

### মূল বৈশিষ্ট্যসমূহ

- **শেয়ার বিনিয়োগ:** বিভিন্ন প্রজেক্টে শেয়ার কিনে মালিকানা অর্জন
- **মাসিক লভ্যাংশ:** প্রজেক্টের মুনাফা থেকে নিয়মিত আয়
- **টাস্ক সিস্টেম:** সামাজিক মিডিয়া টাস্ক করে পয়েন্ট উপার্জন
- **রিওয়ার্ড শপ:** পয়েন্ট দিয়ে বিভিন্ন পুরস্কার নেওয়া
- **উইথড্র সিস্টেম:** bKash এর মাধ্যমে টাকা উত্তোলন
- **রেফারেল বোনাস:** বন্ধুদের রেফার করে বোনাস অর্জন
- **Google লগইন:** এক ক্লিকেই লগইন সুবিধা

## রেজিস্ট্রেশন এবং লগইন

### নতুন অ্যাকাউন্ট তৈরি

#### পদ্ধতি ১: ইমেইল/মোবাইল দিয়ে রেজিস্ট্রেশন

1. **রেজিস্ট্রেশন পেজে যান:**
   - ওয়েবসাইট: `https://buildbargunainitiative.org/register`
   - অ্যাপ: "রেজিস্ট্রেশন" বাটনে ক্লিক করুন

2. **তথ্য পূরণ করুন:**
   ```
   - নাম: আপনার পূর্ণ নাম (ন্যূনতম ২ অক্ষর)
   - ইমেইল: সঠিক ইমেইল ঠিকানা
   - মোবাইল নম্বর: 01XXXXXXXXX ফরম্যাটে (ঐচ্ছিক)
   - পাসওয়ার্ড: ন্যূনতম ৬ অক্ষর
   - রেফারেল কোড: যদি কারো রেফারেন্সে আসেন (ঐচ্ছিক)
   ```

3. **শর্তাবলী মেনে নিন:**
   - সব শর্তাবলী পড়ে নিন
   - চেকবক্সে টিক দিন

4. **রেজিস্ট্রেশন সম্পন্ন করুন:**
   - "রেজিস্ট্রেশন করুন" বাটনে ক্লিক করুন
   - সফল হলে লগইন পেজে রিডাইরেক্ট হবে

#### পদ্ধতি ২: Google অ্যাকাউন্ট দিয়ে রেজিস্ট্রেশন

1. **Google সাইন-ইন বাটনে ক্লিক করুন**
2. **আপনার Google অ্যাকাউন্ট সিলেক্ট করুন**
3. **প্রোফাইল তথ্য দিন:**
   - মোবাইল নম্বর (বাংলাদেশি ফরম্যাট)
   - রেফারেল কোড (ঐচ্ছিক)
4. **প্রোফাইল সম্পন্ন করুন**

### লগইন পদ্ধতি

#### ইমেইল/মোবাইল দিয়ে লগইন

1. **লগইন পেজে যান:**
   - ওয়েবসাইট: `https://buildbargunainitiative.org/login`
   - অ্যাপ: "লগইন" বাটনে ক্লিক করুন

2. **তথ্য দিন:**
   ```
   - ইমেইল অথবা মোবাইল নম্বর
   - পাসওয়ার্ড
   ```

3. **লগইন করুন:**
   - "লগইন করুন" বাটনে ক্লিক করুন

#### Google দিয়ে লগইন

1. **"Google দিয়ে লগইন করুন" বাটনে ক্লিক করুন**
2. **আপনার Google অ্যাকাউন্ট সিলেক্ট করুন**
3. **সরাসরি ড্যাশবোর্ডে প্রবেশ করবেন**

### পাসওয়ার্ড রিসেট

যদি পাসওয়ার্ড ভুলে যান:

1. **লগইন পেজে "পাসওয়ার্ড ভুলে গেছেন?" এ ক্লিক করুন**
2. **রেজিস্টার্ড ইমেইল দিন**
3. **ইমেইল চেক করুন:**
   - পাসওয়ার্ড রিসেট লিঙ্ক পাবেন
   - লিঙ্কটি ১৫ মিনিটের জন্য বৈধ
4. **নতুন পাসওয়ার্ড দিন:**
   - ন্যূনতম ৬ অক্ষর
   - শক্তিশালী পাসওয়ার্ড ব্যবহার করুন
5. **পাসওয়ার্ড রিসেট সম্পন্ন করুন**

### নিরাপত্তা টিপস

✅ **করণীয়:**
- শক্তিশালী পাসওয়ার্ড ব্যবহার করুন

- Google লগইন ব্যবহার করুন (নিরাপদ)

❌ **বর্জনীয়:**
- পাসওয়ার্ড কারো সাথে শেয়ার করবেন না

## ড্যাশবোর্ড ওভারভিউ

### ড্যাশবোর্ড সেকশনসমূহ

লগইন করার পর আপনি ড্যাশবোর্ডে নিম্নলিখিত তথ্যগুলো পাবেন:

#### ১. ব্যালেন্স সারাংশ
```
├─ মোট ব্যালেন্স (টাকা)
├─ উপলব্ধ ব্যালেন্স (উইথড্রযোগ্য)
├─ পেন্ডিং উইথড্র
└─ মাসিক আয়
```

#### ২. বিনিয়োগ সারাংশ
```
├─ মোট বিনিয়োগ
├─ সক্রিয় প্রজেক্ট
├─ মোট শেয়ার সংখ্যা
└─ মোট উপার্জন

#### ৩. পয়েন্ট সারাংশ
```
├─ উপলব্ধ পয়েন্ট

├─ মাসিক অর্জিত পয়েন্ট
└─ লিডারবোর্ড র‍্যাঙ্ক
```

#### ৪. দ্রুত অ্যাকশন
```
├─ নতুন বিনিয়োগ করুন
├─ টাস্ক সম্পন্ন করুন
├─ উইথড্র রিকোয়েস্ট
└─ রিওয়ার্ড রিডিম করুন
```

## প্রজেক্টে বিনিয়োগ

### প্রজেক্ট ব্রাউজ করা

1. **প্রজেক্ট পেজে যান:**
   - মেনু থেকে "প্রজেক্টস" এ ক্লিক করুন
   - অথবা ড্যাশবোর্ড থেকে "বিনিয়োগ করুন" এ ক্লিক করুন

2. **প্রজেক্ট লিস্ট দেখুন:**
   ```
   প্রতিটি প্রজেক্টে যা দেখা যাবে:
   ├─ প্রজেক্টের নাম ও ছবি
   ├─ বর্ণনা
   ├─ মোট মূলধন
   ├─ মোট শেয়ার সংখ্যা
   ├─ শেয়ার প্রতি মূল্য
   ├─ বিক্রি হওয়া শেয়ার
   └─ উপলব্ধ শেয়ার
   ```

3. **প্রজেক্ট বিস্তারিত দেখুন:**
   - যেকোনো প্রজেক্টে ক্লিক করুন
   - বিস্তারিত তথ্য দেখুন

### শেয়ার কেনার নিয়ম

#### ধাপ ১: শেয়ার সিলেক্ট করা

1. **প্রজেক্ট সিলেক্ট করুন**
2. **শেয়ার সংখ্যা দিন:**
   - ন্যূনতম: ১ শেয়ার
   - সর্বোচ্চ: ১০,০০০ শেয়ার (একসাথে)
   - উপলব্ধ শেয়ারের বেশি দেওয়া যাবে না

3. **মোট মূল্য দেখুন:**
   ```
   উদাহরণ:
   শেয়ার প্রতি মূল্য: ৳১০০
   আপনার কেনা শেয়ার: ১০টি
   মোট মূল্য: ৳১,০০০
   ```

#### ধাপ ২: পেমেন্ট মেথড সিলেক্ট করা

**বিকল্প ১: bKash পেমেন্ট**
```
1. bKash সিলেক্ট করুন
2. bKash ট্রানজেকশন আইডি (TxID) দিন
   - ফরম্যাট: ৮-১২ অক্ষর (বড় হাতে)
   - উদাহরণ: 8K7H9M2L
3. আপনার bKash নাম্বার থেকে পাঠানো ট্রানজেকশন আইডি কপি করে দিন
```

**বিকল্প ২: ম্যানুয়াল পেমেন্ট**
```
1. ম্যানুয়াল সিলেক্ট করুন
2. অ্যাডমিন আপনার সাথে যোগাযোগ করবে
3. পেমেন্ট নির্দেশিকা পাবেন
```

#### ধাপ ৩: অনুরোধ জমা দেওয়া

1. **সব তথ্য যাচাই করুন**
2. **"শেয়ার কিনুন" বাটনে ক্লিক করুন**
3. **সফল হলে:**
   - অনুরোধ ID পাবেন
   - স্ট্যাটাস: "অপেক্ষমাণ" (pending)
   - অ্যাডমিন যাচাই করে অনুমোদন দেবে

### অনুরোধ স্ট্যাটাস

| স্ট্যাটাস | অর্থ | পরবর্তী পদক্ষেপ |
|-----------|------|-----------------|
| **অপেক্ষমাণ (Pending)** | অ্যাডমিন অনুমোদনের জন্য অপেক্ষা করছে | অপেক্ষা করুন |
| **অনুমোদিত (Approved)** | শেয়ার আপনার পোর্টফোলিওতে যুক্ত হয়েছে | সার্টিফিকেট ডাউনলোড করুন |
| **প্রত্যাখ্যাত (Rejected)** | অনুরোধ বাতিল হয়েছে | অ্যাডমিন নোট দেখুন, নতুন করে চেষ্টা করুন |

### শেয়ার সার্টিফিকেট

**কীভাবে ডাউনলোড করবেন:**

1. **আমার শেয়ার > শেয়ার রিকোয়েস্টে যান**
2. **অনুমোদিত শেয়ার খুঁজুন**
3. **"সার্টিফিকেট ডাউনলোড" বাটনে ক্লিক করুন**
4. **PDF ফাইল ডাউনলোড হবে**

**সার্টিফিকেটে যা থাকবে:**
- সার্টিফিকেট নম্বর
- প্রজেক্টের নাম
- শেয়ার সংখ্যা
- মোট মূল্য
- আপনার নাম ও ফোন নম্বর
- ক্রয় তারিখ

├─ ওজন (মোট পোর্টফোলিওতে শতাংশ)
├─ মোট উপার্জন
├─ ROI শতাংশ
├─ সর্বশেষ লভ্যাংশের হার
├─ এই মাসের প্রত্যাশিত আয়
└─ মাসিক আয়ের ইতিহাস
```

### পোর্টফোলিও ম্যানেজমেন্ট

#### শেয়ার দেখা

1. **আমার শেয়ার > আমার পোর্টফোলিওতে যান**
2. **সব প্রজেক্টের তালিকা দেখুন**
3. **বিস্তারিত দেখতে প্রজেক্টে ক্লিক করুন**

#### উপার্জন ট্র্যাক করা

1. **আয় > পোর্টফোলিওতে যান**
2. **প্রতিটি প্রজেক্টের মাসিক আয় দেখুন**
3. **মোট আয় এবং প্রজেক্ট ভিত্তিক আয় তুলনা করুন**

### পোর্টফোলিও টিপস

## উপার্জন এবং লভ্যাংশ

### উপার্জন কীভাবে কাজ করে

#### লভ্যাংশ গণনা পদ্ধতি

```
মাসিক লভ্যাংশ = (আপনার শেয়ার × প্রজেক্টের মোট মূলধন × লভ্যাংশের হার) / (মোট শেয়ার × ১০,০০০)

উদাহরণ:
আপনার শেয়ার: ১০০টি
প্রজেক্টের মোট মূলধন: ৳১০,০০,০০০
লভ্যাংশের হার: ১% (১০০ বেসিস পয়েন্ট)
মোট শেয়ার: ১০,০০০টি

লভ্যাংশ = (১০০ × ১০,০০,০০০ × ১০০) / (১০,০০০ × ১০,০০০)
         = ৳১,০০০
```

#### লভ্যাংশ বিতরণ

1. **স্বয়ংক্রিয় বিতরণ:**
   - প্রতি মাসের ১ তারিখে স্বয়ংক্রিয়ভাবে বিতরণ
   - সরাসরি আপনার অ্যাকাউন্টে জমা হয়
   - কোনো ম্যানুয়াল অনুরোধের প্রয়োজন নেই

2. **লভ্যাংশের হার:**
   - অ্যাডমিন প্রতি মাসে হার নির্ধারণ করে
   - প্রজেক্টের আয় অনুযায়ী হার পরিবর্তিত হতে পারে
   - পোর্টফোলিওতে সর্বশেষ হার দেখা যাবে

### উপার্জন সারাংশ

#### মোট ব্যালেন্স

```
মোট ব্যালেন্স = প্রজেক্ট লভ্যাংশ + রেফারেল বোনাস

প্রজেক্ট লভ্যাংশ:
├─ সব প্রজেক্ট থেকে মোট আয়
├─ এই মাসের আয়
├─ গত মাসের আয়
└─ প্রত্যাশিত আয় (এই মাস)

রেফারেল বোনাস:
├─ মোট বোনাস অর্জিত
└─ সক্রিয় বোনাস
```

#### উপার্জন ইতিহাস

1. **আয় > ইতিহাসে যান**
2. **প্রতিটি লেনদেন দেখুন:**
   ```
   ├─ প্রজেক্টের নাম
   ├─ মাস (YYYY-MM)
   ├─ শেয়ার সংখ্যা
   ├─ লভ্যাংশের হার
   └─ উপার্জিত অর্থ
   ```
3. **পেজিনেশন ব্যবহার করুন:**
   - প্রতি পেজে ১০/২৫/৫০টি রেকর্ড
   - পুরনো রেকর্ড দেখতে পেজ পরিবর্তন করুন

### কর্তন এবং ট্যাক্স

⚠️ **গুরুত্বপূর্ণ:**
- লভ্যাংশ থেকে কোনো কর্তন করা হয় না
- ট্যাক্স আপনার দায়িত্বে
