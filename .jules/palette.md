## 2026-04-13 - Add aria-labels to generic UI close buttons
**Learning:** When adding accessibility labels to generic UI elements like dismissible alert close buttons (e.g., ✖) in a localized application, it's important to use established translated phrases (e.g., 'বার্তা বন্ধ করুন' for 'Close message') to match the application interface.
**Action:** Use native language strings for ARIA labels on UI controls whenever the surrounding UI is localized.
